# magazin-de-incaltamintee-v2

O aplicație web simplă pentru vânzarea de încălțăminte. Utilizatorii pot explora produse, vizualiza modele și naviga ușor prin interfață.

# Cum se deschide
Deschide fișierul index.html în browser. Nu este nevoie de instalare.

# Funcționalități
- Răsfoiește o gamă variată de încălțăminte cu imagini de calitate.
- Filtrează produsele după mărime, preț sau brand.
- Adaugă produse în coș și finalizează comanda rapid.
- Primește recomandări personalizate în funcție de preferințe.
